Тернарный оператор

Включает в себя три операнда - ( условие ? Если условие ИСТИННО : Если условие ЛОЖНО ).

Переменной max, присваиваем возвращаемое значение тернарного оператора.

{% highlight javascript %} var a = 1, b = 2, max = 0; // ЕСЛИ: (a > b) ТО: вернуть a ИНАЧЕ: вернуть b; max = (a > b) ? a : b; document.write(max); {% endhighlight %}

Знак "!"

Знак восклицания меняет логическое значение на противоположное.

{% highlight javascript %} var trueVal = true; var falseVal = false; if (trueVal) { document.write("Первый блок if выполнился"); } else if (!falseVal) { document.write("Второй блок else if выполнился"); } else { document.write("Ни один блок не выполнился"); } {% endhighlight %}

If & Else

{% highlight javascript %} var a = prompt("Введите число 22", "0"); if (a == "22") { document.write("Введенное значение верно"); } else { document.write("Введенное значение НЕ верно"); } document.write("
Эта строка выполнится в любом случае"); {% endhighlight %}

Else if

{% highlight javascript %} var a = prompt("Введите слово Hello, или World", "Hello"); if (a == "Hello" || a == 'hello') { document.write("Введенное значение верно и равно Hello"); } else if (a == "World" || a == 'world') { // Выполнится если выражение в перконструкции будет false document.write("Введенное значение верно и равно World"); } else { document.write("Введенное значение НЕ верно"); } document.write("
Эта строка выполнится в любом случае"); {% endhighlight %}

Switch

{% highlight javascript %} var day = prompt("Введите номер дня недели: 1-2-3-4-5-6-7:"); switch (day) { case "1": { document.write("Понедельник"); }; break; case "2": { document.write("Вторник"); }; break; case "3": { document.write("Среда"); }; break; case "4": { document.write("Четверг"); }; break; case "5": { document.write("Пятница"); }; break; case "6": { document.write("Суббота"); }; break; case "7": { document.write("Воскресенье"); }; break; default: { document.write("Вы ввели несуществующий день недели"); }; } {% endhighlight %}

Второй пример использования Switch

{% highlight javascript %} var day = prompt("Введите номер дня недели: 1-2-3-4-5-6-7: ", ""); switch (day) { case "1": case "2": case "3": case "4": case "5": document.write("Этот день недели - Рабочий."); break; case "6": case "7": { document.write("Этот день недели - Выходной."); } break; default: { document.write("Вы ввели несуществующий день недели."); } break; } {% endhighlight %}

Таблица истинности && и ||

Скромная памятка в виде таблицы.

{% highlight javascript %} /* Таблица истинности логического И (&&): true && true = true false && true = false true && false = false false && false = false */ /* Таблица истинности логического ИЛИ (||): true || true = true false || true = true true || false = true false || false = false */ {% endhighlight %}
