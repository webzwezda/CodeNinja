Именна функций даются в стиле UpperCamelCase.

Функция - это блок кода, который можно многократно вызывать на исполнение. При этом функции можно передавать аргументы (параметры) с которыми она будет работать. Аргументы они же - параметры играют роль локальных переменных в теле функции.

Необязательные аргументы

" + msg); } document.write("

"); } // Вызов функций. print("hello"); print("world", 7); {% endhighlight %}

Оператор return

После оператора return функция останавливается и возвращает значение.

{% highlight JavaScript %} function Sum(a, b) { alert('Hello!'); // return прерывает выполнение кода return a + b; } var res = Sum(2, 3); document.write(res); {% endhighlight %}

Колличество аргументов функции

Функция, которая проверяет правильное ли количество аргументов передано. При вызове функции создается специальный масив содержащий в себе аргументы, которые были переданы при вызове. У этого массива есть свойство length, в котором хранится количество аргументов переданых функции.

{% highlight JavaScript %} function f(x, y, z) { // Проверка, правильное ли количество аргументов передано. if (arguments.length != 3) { document.write("

ERROR: Требуется три аргумента при вызове Функция вызвана с " + arguments.length + параметрами!"); } else { document.write("

Правильный вызов функции. Аргументы: "); for (var i = 0; i < arguments.length; i++) document.write(arguments[i] + ", "); } } f(1, 2, 3, 4); // Более трех аргументов - ошибка! f(1, 2, 3); f(1, 2) // Менее трех аргументов - ошибка! {% endhighlight %}
