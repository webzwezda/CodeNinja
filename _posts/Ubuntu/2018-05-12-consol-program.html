youtube-dl

Позволяет скачивать видео с большого количества сайтов, список сайтов впечатляет.

{% highlight bash %} sudo apt-get install youtube-dl {% endhighlight %}

Покажет информацию какие фарматы есть, выбераем из таблицы нужный нам format code.

{% highlight bash %} youtube-dl -F https://youtu.be/moZ4qoFIeUE {% endhighlight %}

Внимание! ключ -f и -F это разные аргументы.

{% highlight bash %} youtube-dl -f formate-code https://youtu.be/moZ4qoFIeUE {% endhighlight %}
