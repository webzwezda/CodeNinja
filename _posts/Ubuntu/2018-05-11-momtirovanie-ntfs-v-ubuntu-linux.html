И так перед нами имеется Linux дистрибутив Ubunt 18.04 LTS и внешний жесткий диск с файлвой системой NTFS.

Как узнать какой диск монтировать?

Команда для просмотра информации о всех подключенных дисках в Ubuntu

Вводим команду без подключенного диска потом подключакм диск снова вводим команду и смотрем что изменилось. Так и узнаем какой диск нужно монтировать.

{% highlight bash %} sudo blkid {% endhighlight %}

Так я узнаю что монтировать нужно раздел /dev/sdb1 у которого файловая система NTFS

Команда монтирования NTFS диска в Ubuntu

Эта команда доступна сразу после установки Ubuntu

{% highlight bash %} sudo ntfsfix /dev/sdb1 {% endhighlight %}
