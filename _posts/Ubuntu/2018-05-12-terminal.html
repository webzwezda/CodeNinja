Выполнение дальнейших команд от имени root

{% highlight bash %} sudo -s sudo -i {% endhighlight %}

Пути к файлам из терминала

{% highlight bash %} dir / - Корневая директория dir ~/ - Домашняя директория pwd - Покажет полный путь где находимся {% endhighlight %}

Создание файла в linux ubuntu

{% highlight bash %} touch text.txt echo >>text.txt {% endhighlight %}

Добавим в этот файл какой либо текст

{% highlight bash %} echo >text.txt pidorov Способ 2 echo -e "pidorov" >>text.txt ls -l >text.txt {% endhighlight %}

Вывод содержимого файла в терминал

{% highlight bash %} cat file.txt {% endhighlight %}

Удаление папки с файлами в ubuntu

{% highlight bash %} rm -rf dir_name {% endhighlight %}

Поиск файлов через терминал ubuntu

{% highlight bash %} find -name text* * - Указывает на любую последовательность символов {% endhighlight %} {% highlight bash %} apt moo or apt moo moo or apt moo moo moo {% endhighlight %}

Имя машины

Команда покажет имя машины из под которой работаем

{% highlight bash %} uname -a {% endhighlight %}

Права на файл

{% highlight bash %} sudo chmod -R 777 * - Все папки в папке sudo chown root:root text.txt {% endhighlight %}
