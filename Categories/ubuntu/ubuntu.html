Шрифты от Microsoft

Не только шрифты но еще и кодеки.

{% highlight bash %} sudo apt install ubuntu-restricted-extras {% endhighlight %}

Добавляем в контекстное меню опцию создания документов

Вводить последовательно по одной

{% highlight bash %} XDG_TEMPLATES_DIR=$(xdg-user-dir TEMPLATES) cd "$XDG_TEMPLATES_DIR" touch 'New Text File.txt' && touch 'New Word File.doc' && touch 'New Excel Spreadsheet.xls' {% endhighlight %}

Решение проблемы с кириллицей в текстовом редакторе Gedit

Внимание!Это одна команда и должна быть прописанна в строчку

{% highlight bash %} gsettings set org.gnome.gedit.preferences.encodings candidate-encodings "['UTF-8', 'WINDOWS-1251', 'KOI8-R', 'CURRENT', 'ISO-8859-15', 'UTF-16']" {% endhighlight %}

Установка дополнительной поддержки архиваторов

{% highlight bash %} sudo apt install p7zip-rar rar unrar unace arj cabextract {% endhighlight %}

Gnome Tweak Tool

{% highlight bash %} sudo apt install gnome-tweak-tool {% endhighlight %}

Московское время

Установим отображение московского времени. {% highlight bash %} sudo apt install chrome-gnome-shell {% endhighlight %}

Установим браузерное расширение через Firefox Ссылка

В директории home создадим файл people.json

Поместим туда наш код.

| Примеры | Time Zone | Видео инструкция |

{% highlight json %} [ { "name": "Moscow", "avatar": "/home/syshka/Pictures/moscow.jpeg", "city": "MSC", "tz": "Europe/Moscow" } ] {% endhighlight %}

Настройка OpenVPN

{% highlight bash %} sudo apt install network-manager-openvpn {% endhighlight %}

Далее нам понадобится файл настройки Config
